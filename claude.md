# あやとりアプリ (String Figures App)

## プロジェクト概要
日本の伝統的なあやとり（String Figures）を学習できるReact Native Expoアプリです。TypeScriptで開発され、直感的なUIでさまざまなあやとりの作り方を紹介します。

## 技術スタック
- **Framework**: React Native Expo
- **Language**: TypeScript
- **Template**: blank-typescript

## アプリ機能

### 1. TOPページ（メイン画面）
- **ヘッダー**: 「あやとり」タイトルとメニューボタン（3点ドットアイコン）
  - **ドロップダウンメニュー**:
    - 言語設定（日本語）
    - 動画の字幕設定（あり/なし）
    - 購入情報を復元
- **フィルター機能**: 難易度別フィルタリング
  - かんたん（緑色）
  - ふつう（黄色）
  - むずかしい（オレンジ色）
- **あやとり一覧**: グリッドレイアウトでカード表示
  - 各カードには作品の写真、名前、難易度アイコンを表示
  - タップすると詳細ページ（ボトムシート）を表示

### 2. 詳細ページ（ボトムシート）
- **表示方式**: TOPページ上にボトムシート風のオーバーレイ表示
- **コンテンツ**:
  - 予告編動画を自動でループ再生
  - 再生ボタン（動画ページへ遷移）
  - サムネイル画像
  - 作品名（例：「イヌイットの網」）
  - 難易度表示
  - 説明文
- **アクション**: 再生ボタンタップで動画ページへ遷移

### 3. 動画ページ（90回転させてぱっと見ランドスケープのように表示）
- **機能**:
  - 画面の右側3/4程度の大きな動画プレイヤー
  - 動画コントロール（再生/一時停止、前後スキップ、巻き戻し）
  - 再生速度調整（1.0x表示、速度変更可能）
  - 進捗バー
  - 字幕表示（手順説明）
  - 戻るボタン
  - ブックマークボタン

## UI/UXデザイン仕様

### カラーパレット
- **プライマリ**: 温かみのあるベージュ/クリーム色
- **アクセント**: 青色（紐の色）
- **難易度色**:
  - 簡単: 緑色
  - 普通: 黄色  
  - 難しい: オレンジ色

### タイポグラフィ
- **日本語フォント**: 読みやすい明朝体またはゴシック体
- **階層**: タイトル > 作品名 > 説明文

### レイアウト
- **TOPページ**: 2列グリッドレイアウト
- **詳細ページ**: 中央配置のボトムシート
- **動画ページ**: フルスクリーンランドスケープ

## 技術要件

### ナビゲーション
- TOPページ ⇔ 詳細ページ: ボトムシートの表示/非表示
- 詳細ページ → 動画ページ: 画面遷移

## 開発フェーズ

### Phase 1: 基本画面実装
1. TOPページのレイアウト作成
2. あやとり一覧の表示
3. フィルター機能実装

### Phase 2: 詳細ページ実装
1. ボトムシートコンポーネント作成
2. 詳細情報の表示
3. 画面遷移の実装

### Phase 3: 動画ページ実装
1. ランドスケープレイアウト作成
2. 動画プレイヤー実装

### Phase 4: 機能拡張
1. ブックマーク機能
2. 検索機能
3. ユーザー設定
4. 多言語対応（メニューから切り替え）
5. 字幕表示設定
6. アプリ内課金・購入復元機能

## 必要なライブラリ（推定）
- `expo-av`: 動画再生
- `react-native-reanimated`: アニメーション（ボトムシート）
- `react-native-gesture-handler`: ジェスチャー操作
- `@expo/vector-icons`: アイコン表示
- `expo-localization`: 多言語対応
- `react-i18next`: 国際化ライブラリ
- `expo-store-review`: アプリ内課金・購入復元
- `@react-native-async-storage/async-storage`: ローカル設定保存

## アプリ設定機能
